## 主要な発見のサマリー

### 5つの根本原因
1. ❌ angle情報の欠落
2. ❌ Bisector距離計算式の誤り (`R/cos` → `R×tan`)
3. ❌ テーパー専用補正式の不在
4. ❌ Bisectorの直線への誤用（形状の歪み）⭐ 最重要
5. ❌ 補正量の二重適用

### 修正結果
- **誤差**: -0.428mm → **0.000mm** ✅
- **角度**: 歪む → **保持** ✅
- **理論**: 不一致 → **完全一致** ✅

## 理論的背景

### ユーザーの記事（正解）
- 上りテーパー（直径減少）: `fz = R × (1 - tan(θ/2))`
- 下りテーパー（直径増加）: `fz = R × (1 + tan(θ/2))`

### 重要な発見
```
fz は合計補正量（垂直オフセットを含む）
テーパー線では P座標にオフセットを含めない
→ n = {nx: 0, nz: 0}
→ oz = pz - fz
```

## よくある質問（FAQ）

### Q: なぜBisector Methodは間違っていたのか？
A: Bisectorは「コーナー（交点）」用の手法。直線セグメント、特にテーパー線には不適切。始点と終点を独立に計算すると直線が歪む。

### Q: R/cos(θ/2) と R×tan(θ/2) の違いは？
A:
- `R/cos(θ/2)`: 無限直線の幾何学的交点距離（理論値）
- `R×tan(θ/2)`: 実際の工具中心軌跡の距離（実用値）

### Q: 「上り/下り」はどう判定する？
A: **直径の変化**で判定（Z座標ではない）
- 上り: 直径減少 (`endX < startX`)
- 下り: 直径増加 (`endX > startX`)

### Q: fzは追加の補正量？
A: いいえ。**合計補正量**です。垂直オフセットを含む完全な値。だからP座標にオフセットを含めない。

### Q: この修正は他のケースにも影響する？
A: テーパー線（angle ≠ 0, 90）のみ影響。垂直線、水平線、円弧は影響なし。
