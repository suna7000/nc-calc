## Bisectorの問題

### 問題: 独立したノード計算

```
        テーパー線        垂直線
          ╲               │
           ╲              │
            ╲             │
        ●────●────●      ← 3つのノード
       N0   N1   N2

N0: テーパーの始点
    → テーパーの垂直方向にオフセット

N1: テーパーと垂直線の接続点
    → 両者のBisector方向にオフセット ← 問題！

N2: 垂直線の終点
    → 垂直線の垂直方向にオフセット


【結果】
           ╲ ←N0
            ╲
             ╲
          ━━━●━━━  ← N1 (Bisectorで曲がる)
               │   ← N2
               │

テーパー線が曲がってしまう！
```

### 解決: テーパー線は一様にオフセット

```
        テーパー線        垂直線
          ╲               │
           ╲              │
            ╲             │
        ●────●────●

すべてのノードで同じ方向:
n = { nx: 0, nz: 0 }  ← オフセットなし！

補正は dz だけで適用:
fz = R × (1 - tan(θ/2))


【結果】
           ╲ ←N0
            ╲
             ╲
              ●────  ← N1 (まっすぐ！)
               │   ← N2
               │

テーパー角度が保持される！
```

## 補正量の二重適用問題

### 修正前: 垂直オフセット + dz

```
【ステップ1】P座標の計算

    ワーク表面 (30°テーパー)
         ╲
          ╲   ↑ nz × R = -0.4mm
           ╲  ↑ (垂直オフセット)
         ●──●  ← P座標
        幾何  pz = -46 - 0.4 = -46.4


【ステップ2】O座標の計算

         ●  ← P座標 (pz = -46.4)
         ↓
         ↓ dz = 0.586mm
         ↓
         ●  ← O座標 (oz = -46.4 - 0.586 = -46.986)

【結果】
合計補正 = 0.4 + 0.586 = 0.986mm ❌
期待値 = 0.586mm
誤差 = 0.4mm (垂直オフセットが余分)
```

### 修正後: fz だけを適用

```
【ステップ1】P座標の計算

    ワーク表面 (30°テーパー)
         ╲
          ╲
           ╲  ← オフセットなし
         ●
        幾何  pz = -46 (幾何座標そのまま)


【ステップ2】O座標の計算

         ●  ← P座標 (pz = -46)
         ↓
         ↓ dz = 0.586mm (fz)
         ↓
         ●  ← O座標 (oz = -46 - 0.586 = -46.586)

【結果】
合計補正 = 0.586mm ✅
期待値 = 0.586mm ✅
誤差 = 0.000mm
```

## Bisector距離計算式の比較

### R/cos(θ/2) vs R×tan(θ/2)

```
【2本の直線が90°で交わる場合】

        直線2
         │
         │ ↗ Bisector (45°)
         │↗
      ───┼───  直線1
         R


■ 修正前: R/cos(θ/2)

  cos(45°) = 0.707
  dist = 0.8 / 0.707 = 1.131mm

  交点位置:
         │
         │  ↗
         │   ↗ 1.131mm ← 遠すぎる
         │    ↗
      ───┼──────●
         R


■ 修正後: R×tan(θ/2)

  tan(45°) = 1.0
  dist = 0.8 × 1.0 = 0.8mm

  交点位置:
         │
         │  ↗
         │  ↗ 0.8mm ← 正しい
         │ ↗
      ───┼───●
         R
```

### 数値比較表

| 角度θ | R/cos(θ/2) | R×tan(θ/2) | 差分 | 誤差率 |
|-------|------------|------------|------|--------|
| 30° | 1.035R | 0.268R | +0.767R | +286% |
| 60° | 1.155R | 0.577R | +0.578R | +100% |
| 90° | 1.414R | 1.000R | +0.414R | +41% |
| 120° | 2.000R | 1.732R | +0.268R | +15% |

R=0.8mmの場合、90°で 0.331mm の誤差！

## 「上り/下り」判定の間違い

### 誤った判定: Z座標で判定

```
❌ const isDescending = (seg.endZ < seg.startZ)

    ●───────●
   Z-45.653 Z-46

   Z-46 < Z-45.653 → true
   → 「下りテーパー」と判定
   → fz = R × (1 + tan(θ/2)) = 1.014mm

   しかし実際は「上りテーパー」（直径が減少）
```

### 正しい判定: 直径で判定

```
✅ const isDiameterIncreasing = (seg.endX > seg.startX)

    ●───────●
   X60     X59.6

   59.6 < 60 → false (増加していない)
   → 「上りテーパー」と判定
   → fz = R × (1 - tan(θ/2)) = 0.586mm ✅
```

### 旋盤座標系での「上り/下り」

```
【上りテーパー】直径が減少
     ╲
      ╲  ← ワーク表面が「上がる」
       ╲
    細い  太い
    ←───────

    直径: 60 → 59.6 (減少)
    fz = R × (1 - tan(θ/2))


【下りテーパー】直径が増加
       ╱
      ╱  ← ワーク表面が「下がる」
     ╱
    太い  細い
    ───────→

    直径: 59.6 → 60 (増加)
    fz = R × (1 + tan(θ/2))
```

## fz の物理的意味

### fz は合計補正量

```
        工具中心
           ●
          /│╲
       R/ │ ╲
        /  │fz╲
       /___|___╲
      θ/2      30°テーパー


【分解すると】
- 垂直方向成分
- 接線方向成分
- 角度補正成分

これらの合計が fz = R × (1 - tan(θ/2))


【重要】
fz は「追加の補正」ではない
「完全な補正量」である

だから:
  工具中心Z = 幾何Z - fz

これだけで完結！
```
