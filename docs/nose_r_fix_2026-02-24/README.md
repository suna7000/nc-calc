## 5つの根本原因

1. **angle情報の欠落** - shape.ts で angle が失われていた
2. **Bisector距離計算式の誤り** - `R/cos(θ/2)` → `R×tan(θ/2)`
3. **テーパー専用補正式の不在** - `fz = R×(1±tan(θ/2))` を実装
4. **Bisectorの直線への誤用** ⭐ 最重要 - セグメントが歪む
5. **補正量の二重適用** - 垂直オフセット + fz が適用されていた

## 最も重要な発見

```
Bisector Method は「コーナー（交点）」用の手法
直線セグメント、特にテーパー線には不適切

テーパー線の正しいアプローチ:
1. P座標 = 幾何座標（オフセットなし）
   → n = {nx: 0, nz: 0}
2. O座標 = P - fz（記事の数式を直接適用）

シンプルで直接的！
```
