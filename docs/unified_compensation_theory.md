# ノーズR補正 統一計算理論

## 目的

本ドキュメントは、ノーズR補正の**物理的本質**から出発し、数学（幾何学）と機械工学（CNC加工）の知見を統合して正確な計算方法を記述する。

---

## 0. ノーズR補正とは何か

### 0.1 3つの重要な点

NC旋盤の工具先端は**完全な鋭角ではなく、ノーズRという微小な円弧**を持つ。このため、工具位置の制御には以下の3点の関係を理解する必要がある。

```
                  工具中心 P
                    ●
                   /|\
                  / | \
                 /  |  \  R（ノーズ半径）
                /   |   \
               /    |    \
              ╭─────●─────╮
             /   切削点    \    ← ワーク表面上の接点（実際に削れる位置）
            ╰───────────────╯
                    ↓
              仮想刃先点 O       ← NCプログラムの座標基準点
        （チップ番号で位置が決まる）
```

| 点 | 記号 | 意味 |
|:---:|:---:|:---|
| 仮想刃先点 | O | NCプログラムの座標。チップ番号で決まる位置 |
| 工具中心 | P | ノーズRの円の中心。O から V_tip だけ離れた位置 |
| 切削点 | — | ワーク表面上の接点。実際に切削が行われる位置 |

### 0.2 なぜ補正が必要か

**水平面（外径面）や垂直面（端面）では、仮想刃先点Oと切削点が一致するため補正は不要。**

しかし**テーパー面では、仮想刃先点Oと実際の切削点がずれる**。これがノーズR補正が必要な根本理由。

```
外径面（θ=90°）           端面（θ=0°）            テーパー面（0°<θ<90°）
  │                      ─────                    ╲
  │← 切削点=O            ↓切削点=O                  ╲← 切削点
  │  （ずれなし）          （ずれなし）                  ╲
  │                                                  ↓ O（ずれあり！）
```

テーパー面では、仮想刃先点Oを基準にプログラムすると実際の切削点との間にずれが生じる。このずれを補正するのが **fz（Z方向）** と **fx（X方向）** の補正量。

### 0.3 補正の2つのアプローチ

| アプローチ | 方法 | 使用場面 |
|:---|:---|:---|
| **CNC自動補正 (G41/G42)** | コントローラが2-3ブロック先読みし、オフセット曲線の交点を自動計算 | CNC機がG41/G42をサポートする場合 |
| **手動補正 (fz/fx公式)** | プログラマが各座標点にfz, fxを加算してプログラム | G41/G42を使わない場合（本アプリの対象） |

## 1. テーパー補正公式: fz と fx

### 1.1 fz — Z方向の補正量

テーパー角θの面において、仮想刃先点と実際の切削点のZ方向のずれ：

**直径減少方向（上りテーパー）**:
```
fz = R × (1 - tan(θ/2))
```

**直径増加方向（下りテーパー）**:
```
fz = R × (1 + tan(θ/2))
```

#### 判定基準

**直径の増減で判定する（Z座標ではない）**:
```
直径減少: endX < startX → fz = R × (1 - tan(θ/2))
直径増加: endX > startX → fz = R × (1 + tan(θ/2))
```

#### 幾何学的導出

テーパー角θの面に対して、ノーズR円の接線を引くと：

```
          P（工具中心）
          ●
         /|╲
        / | ╲  R
       /  |  ╲
      /   | θ/2╲
  接点●───┼───●  仮想刃先点の高さ
      ↑       ↑
     切削点    Oの投影位置

  Z方向のずれ = R - R×tan(θ/2) = R×(1 - tan(θ/2))
```

半角公式 `tan(θ/2) = (1 - cosθ) / sinθ` を用いて、ノーズR円の中心Pから仮想刃先点Oまでのベクトルと、Pから切削点（接点）までのベクトルのZ成分の差を取ると：

- 上りテーパー: `fz = R × (1 - tan(θ/2))`
- 下りテーパー: `fz = R × (1 + tan(θ/2))`

### 1.2 fx — X方向の補正量（直径値）

テーパー角θの面における、仮想刃先点と実際の切削点のX方向のずれ：

```
fx = 2R × (1 - tan(φ/2))     ただし φ = 90° - θ
```

ここで `φ = 90° - θ` はテーパー角の余角。直径値で表記（半径値なら `R × (1 - tan(φ/2))`）。

#### 幾何学的意味

fzがZ方向のずれなら、fxはX方向のずれ。テーパー角θに対して：
- θが小さい（ほぼ端面） → fzが大きく、fxが小さい
- θが大きい（ほぼ外径面） → fzが小さく、fxが大きい
- θ = 45° → fz(半径値) = fx(半径値)（対称）

### 1.3 特殊ケース

| 面 | θ | fz | fx(直径値) | 説明 |
|:---:|:---:|:---:|:---:|:---|
| 端面 | 0° | R | 0 | Z方向のみ補正 |
| 外径面 | 90° | 0 | 2R | X方向のみ補正 |
| 45°テーパー | 45° | R(1-tan22.5°) | 2R(1-tan22.5°) | fz半径値 = fx半径値 |

端面(θ=0°)では fz = R, fx = 0 → Z方向のみ補正。
外径面(θ=90°)では fz = 0, fx = 2R → X方向のみ補正。
これは物理的に正しい（0.2節の図と整合する）。

### 1.4 代表値テーブル（R = 0.4mm）

| 角度θ | fz(上り) | fz(下り) | fx(直径値) | φ(=90°-θ) |
|:---:|:---:|:---:|:---:|:---:|
| 0° | 0.400 | 0.400 | 0.000 | 90° |
| 15° | 0.347 | 0.453 | 0.186 | 75° |
| 30° | 0.293 | 0.507 | 0.338 | 60° |
| 45° | 0.234 | 0.566 | 0.469 | 45° |
| 60° | 0.169 | 0.631 | 0.586 | 30° |
| 90° | 0.000 | 0.800 | 0.800 | 0° |

**検算**: θ=45°で fx(半径値) = R(1-tan(22.5°)) = 0.234 = fz(上り) → 対称性を確認。

## 2. 円弧セグメントの補正

### 2.1 基本原理

ノーズR円の中心Pは、ワーク表面から常にR離れた軌跡を描く。円弧の場合、この軌跡は**同心円弧**になる。

| 円弧タイプ | 補正後半径 | 理由 |
|:---:|:---:|:---|
| 凸円弧（角R/kaku-r） | R_arc + R_nose | 工具が外側を通る → 半径増大 |
| 凹円弧（隅R/sumi-r） | R_arc - R_nose | 工具が内側を通る → 半径縮小 |

**中心座標は変わらない**（同心円弧のため）。

### 2.2 ガウジング条件

凹円弧で `R_arc < R_nose` の場合、補正後半径が負になり物理的に切削不可能（ガウジング）。

```
制約: R_arc > R_nose （凹円弧の場合）
```

## 3. コーナーノードの bisector 計算

### 3.1 二等分線の距離公式

2つのセグメントが角度αで交わるコーナーでは、法線ベクトルの二等分線方向に工具中心をオフセットする。

```
dist = R / cos(α/2)
```

この公式は**2本のオフセット線の交点**（二等分線上の距離）を求める。

#### R×tan(α/2) との違い

- `R × tan(α/2)` = コーナー頂点から**ライン沿い**の接線点までの距離
- `R / cos(α/2)` = コーナー頂点から**二等分線沿い**のオフセット交点までの距離

```
         R/cos(α/2)
  頂点 ●──────────────● P（オフセット交点）
        ╲             │
  R×tan  ╲            │ R（オフセット距離）
          ╲ α/2       │
           ╲──────────●  接線点
```

#### 検証: 90°コーナーでの確認

```
90°コーナー（垂直→水平）:
- オフセット線は (R, 0) から上方、(0, R) から右方に延びる
- 交点 = (R, R)、距離 = R√2 = R/cos(45°) ✓
- R×tan(45°) = R ≠ R√2  ← 二等分線上の距離としては不正確
```

#### 手書き検証での改善

IMG_1423 C0.2面取り接続点の水平線終点X:
- R×tan: X78.254（手書きX78.622との差 0.368mm）
- **R/cos: X78.663（手書きX78.622との差 0.041mm）** ← 大幅改善

### 3.2 接線連続ノードでの挙動

shape.tsが生成するプロファイルでは、直線→凸円弧の接続は接線連続（G1連続）。
このとき `α ≈ 0` となるため：

```
dist = R / cos(0) = R    → P = ref + n × R（正しい単純オフセット）
```

R/cos公式では接線連続ノード（α=0）で自然に `dist = R` となり、理論的に正しい結果を得る。
（旧R×tan公式では `dist ≈ 0` となり、pToOの条件付きdzで補完が必要だった）

### 3.4 凸/凹コーナーの判定

- **凸コーナー**: 工具が外側を回る → 補正半径が大きくなる
- **凹コーナー**: 工具が内側に入る → 補正半径が小さくなる

## 4. 現行実装の構造

### 4.1 三重モデルの設計判断

現行実装(`noseRCompensation.ts`)は3つの計算モデルを使い分けている：

| モデル | 対象 | 計算方法 |
|:---|:---|:---|
| **fz公式（テーパー終点）** | テーパー→次セグメント接合 | `P.z = ref.z - fz`、`P.x = ref.x + nextNormal.nx × R` |
| **fz公式（テーパー始点）** | 前セグメント→テーパー接合 | `P.z = ref.z - fz`、`P.x = ref.x + prevNormal.nx × R` |
| **bisector法** | 非テーパーのノード | `P = ref + bisector × R/cos(α/2)`、`O = P - V_tip` |

#### テーパー始点/終点の特徴

- **Z成分**: 両方とも同じfz公式を使用（ascending: `R(1-tan(θ/2))`, descending: `R(1+tan(θ/2))`）
- **X成分**: テーパー自身の法線ではなく、**隣接セグメントの法線**を使用
  - テーパー始点: 前セグメントの終了法線（prevNormal）
  - テーパー終点: 次セグメントの開始法線（nextNormal）
- **dz**: テーパー公式使用時は常に dz=0（fzに全Z補正が含まれる）

この設計により、手書き計算値との完全一致が実現（IMG_1423: テーパー始点X62 Z-44.508、終点X59.6 Z-46.586）。

### 4.2 pToO の条件付き dz

```typescript
dz = isConvex ? 0 : noseR
```

この条件分岐は bisector法の特性（接線連続ノードで dist≈0 になる問題）に対する**正しい対応**であり、ワークアラウンドではない。

- **凸円弧(角R)**: bisector計算が十分なオフセットを含むため dz=0
- **凹円弧(隅R)/直線**: 追加の Z方向オフセットが必要

### 4.3 V_tip テーブル

仮想刃先点O と工具中心P の関係: `P = O + V_tip`

| Tip | V_tip.x(半径値) | V_tip.z | 典型用途 |
|:---:|:---:|:---:|:---|
| 3 | +R | +R | 外径/前/右勝手 |
| 2 | -R | -R | 内径/前/右勝手 |
| 1 | -R | +R | 内径/奥 |
| 4 | +R | -R | 外径/奥 |
| 8 | +R | 0 | 端面/X方向逃げ |

### 4.4 分離原則

計算は2段階に分離する:

```
Step 1: ワーク形状座標の計算（shape.ts）
        → 図面値のR/Cを使用、ノーズRは考慮しない

Step 2: ノーズR補正の適用（noseRCompensation.ts）
        → Step 1の結果にノーズR補正を適用
```

**Step 1でノーズRを加算してはならない**（二重補正の原因）。

## 5. 既知の課題

### 5.1 テーパー→隅R接続の誤差

テーパー線から凹円弧（隅R）への接続で約0.030mmの誤差が存在。
`fz_effective = fz × cos(θ)` の経験的補正で対応しているが、数学的導出が不完全。

### 5.2 接線連続ノードのZ不連続

直線→凸円弧の接続で ΔZ = noseR の不連続が発生。
同一ノードのP座標から pToO で異なるdzが適用されることが原因：
- 直線側: dz = noseR
- 凸円弧側: dz = 0

実用上は問題ないが、理論的には改善の余地がある。
