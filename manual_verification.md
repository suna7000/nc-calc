# 教科書式の手計算検証

## ユーザーの手書き期待値との比較

### 入力条件
- 元の図面座標: X59.6 Z-46
- 前の点: X60 Z-45.653
- ノーズR: 0.8mm
- ツールチップ番号: 3（外径/前向き）

### テーパー角度の計算
```
ΔX（直径） = 60 - 59.6 = 0.4mm
ΔX（半径） = 0.4 / 2 = 0.2mm
ΔZ = -46 - (-45.653) = -0.347mm

tan(θ) = |ΔX半径| / |ΔZ|
tan(θ) = 0.2 / 0.347 = 0.5764
θ = atan(0.5764) = 29.96° ≈ 30° ✓
```

### 教科書式による補正量計算

#### Z方向補正（fz）
```
fz = R × (1 - tan(θ/2))
fz = 0.8 × (1 - tan(15°))
fz = 0.8 × (1 - 0.26795)
fz = 0.8 × 0.73205
fz = 0.58564mm
```

**手計算: fz = 0.586mm** ← ユーザーの手書き期待値と一致！

#### X方向補正（fx）
```
φ = 90° - θ = 90° - 30° = 60°
fx = 2R × (1 - tan(φ/2))
fx = 2 × 0.8 × (1 - tan(30°))
fx = 1.6 × (1 - 0.57735)
fx = 1.6 × 0.42265
fx = 0.67624mm
```

**手計算: fx = 0.676mm**

### 補正座標の計算

テーパー方向の判定:
- ΔX半径 = -0.2（負）
- **下りテーパー**（X減少方向）

下りテーパーの場合の適用方法:
```
X' = X + fx = 59.6 + 0.676 = 60.276mm
Z' = Z - fz = -46 - 0.586 = -46.586mm
```

**手計算結果:**
- X' = 60.276mm
- Z' = -46.586mm

### 実装値との比較

実装値（formula_validation.test.tsの出力）:
- X = 60.275mm
- Z = -46.586mm

誤差:
- X誤差: 60.275 - 60.276 = -0.001mm
- Z誤差: -46.586 - (-46.586) = 0.000mm

### ユーザーの手書き期待値との比較

ユーザーの手書きメモ:
- Z補正後: **-46.586mm**

手計算結果:
- Z補正後: **-46.586mm**

実装値:
- Z補正後: **-46.586mm**

**完全一致！**

## 45°テーパーの検証

### 入力条件
- 元の座標: X100 Z0 → X120 Z-10
- tan(45°) = 1.0 なので ΔX半径 = ΔZ

### 角度計算
```
ΔX半径 = (120 - 100) / 2 = 10mm
ΔZ = -10mm
tan(θ) = 10 / 10 = 1.0
θ = 45° ✓
```

### 補正量計算
```
fz = R × (1 - tan(22.5°))
fz = 0.8 × (1 - 0.41421)
fz = 0.8 × 0.58579
fz = 0.46863mm

φ = 90° - 45° = 45°
fx = 2R × (1 - tan(22.5°))
fx = 1.6 × 0.58579
fx = 0.93726mm
```

### 補正座標（上りテーパー: X増加）
```
X' = 120 - 0.937 = 119.063mm
Z' = -10 - 0.469 = -10.469mm
```

実装値:
- X = 119.063mm ✓
- Z = -10.469mm ✓
